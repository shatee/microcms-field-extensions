# microcms-field-extensions

microCMS の拡張フィールド

## 拡張フィールド一覧

* time-range
  * URL: https://shatee.github.io/microcms-field-extensions/time-range/
* address-from-zipcode-description-pref
  * URL: https://shatee.github.io/microcms-field-extensions/address-from-zipcode-description-pref/
  * https://github.com/microcmsio/microcms-field-extension/tree/main/examples/address-from-zipcode を元に description に都道府県を入れたもの。検索用。
## 開発

### インストール

```
pnpm i
```

### ローカルビルド

```
pnpm dev
```

### ビルド

```
pnpm build
```
